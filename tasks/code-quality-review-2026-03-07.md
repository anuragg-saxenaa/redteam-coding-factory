## 1. CRITICAL: RedTeamFactory Missing API Methods

**Problem:** RedTeamFactory class does not expose `submitTask` method, causing test failures and breaking CLI functionality.

**Details:**
- Test `redteam-factory.test.js` fails with "factory.submitTask is not a function"
- CLI (`src/cli.js`) calls `factory.submitTask(repo, task)` and `factory.submitCrossRepoTask(task)`
- These methods are not defined in RedTeamFactory class

**Fix Required:**
Add delegation methods to RedTeamFactory:

```javascript
/**
 * Submit a task to a specific repo
 */
submitTask(repoName, task) {
  if (!this.orchestrator) {
    throw new Error('Factory not initialized. Call initialize() first.');
  }
  return this.orchestrator.submitTask(repoName, task);
}

/**
 * Submit a cross-repo task
 */
submitCrossRepoTask(task) {
  if (!this.orchestrator) {
    throw new Error('Factory not initialized. Call initialize() first.');
  }
  return this.orchestrator.submitCrossRepoTask(task);
}

/**
 * Get task history
 */
getTaskHistory() {
  return this.taskLog;
}

/**
 * Get result history
 */
getResultHistory() {
  return this.resultLog;
}
```
